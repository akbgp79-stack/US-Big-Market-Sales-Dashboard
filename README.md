# Sales Performance Dashboard (Power BI)

## Project Overview
This project analyzes sales performance across regions, states, and product categories using Power BI.

## Dashboard Features
- KPI Cards (Sales, Profit, Quantity, Profit Margin)
- Sales by Region
- Profit by Sub Category
- Top 10 States by Profit
- Bottom 10 States by Profit
- Sales Treemap by Category

## Tools Used
- Power BI
- Power Query
- DAX

- Texas showed the highest loss.
- Phones and Chairs were the top revenue generating categories.
